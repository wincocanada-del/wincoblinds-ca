# Winco Blinds Website Starter

New website starter folder for `wincoblinds.ca`.

## Structure

- `index.html` — Home page starter
- `about.html` — About page starter
- `services.html` — Services page starter
- `products.html` — Products page starter
- `dual-shades.html` — Dual Shades page starter
- `roller-shades.html` — Roller Shades page starter
- `motorized-blinds.html` — Motorized Blinds page starter
- `guides.html` — Guides page starter
- `gallery.html` — Gallery page starter
- `faq.html` — FAQ page starter
- `reviews.html` — Reviews page starter
- `blog.html` — Blog page starter
- `contact.html` — Contact page starter
- `app/index.html` — Winco app gateway page for `/app`
- `assets/css/style.css` — Main styling
- `assets/js/main.js` — Navigation script
- `assets/js/products-data.js` — Starter product data file
- `assets/images/` — Image folders

## Recommended workflow

1. Open this folder in VS Code.
2. Run with Live Server.
3. Edit page by page with Codex.
4. Commit to GitHub repo named `wincoblinds-ca`.
5. Deploy with GitHub Pages, Netlify, or Vercel.

## Codex instruction example

Do not rebuild the whole website. Only update `index.html` and `assets/css/style.css`. Keep the current folder structure. Improve the Home page design in a modern minimal premium style for Winco Blinds.
